Description

Cisco verifies the earner of this badge successfully completed the Enterprise Networking, Security and Automation course and achieved this student level credential. Earning Criteria

  • Accumulated up to 15 hours of hands-on labs using Cisco hardware and/or Cisco Packet Tracer tool.
  • Passed instructor-led assessments and final examination.
